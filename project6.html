Little Bits Cuckoo Clock

Physical Computing

Objective

The objective of this project was to protoype an installation or game using Little Bits for use in the public setting. We decided to create a cuckoo clock.

 

Creation Process

 

Little Bits Equipment

First we organized and cataloged what Little Bits pieces we had available. We worked with the main kit to create the functions of the clock. We planned to use the servo motor to move the bird, which we would 3D print, and LED screen for the clock. We also used the speaker to play sound every time the clock reached a new hour.

 

Creating Music

Our next step was to create the music for the LittleBits speaker. Using sheet music for Heart and Soul, we took the main notes and programmed them into the Little Bits.

 

Painting the Birdhouse

We used acrylic paint on a wooden birdhouse.

 

3D Printing the Bird

We used a 3D Printer to create the bird, and then painted it with acrylic paint.

 

Assembly

Finally, we assembled all the pieces together.
