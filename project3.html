Cat in the City: A Creative Coding Game

Game Design/Creative Coding

Objective

The objective of this project was to design and develop a game using the p5.js Javascript library. This is an open-source library for creative coding. In particular, I referenced the p5.play library, which provides animations, input, and collision functions for games and gamelike applications.

 

Process

After my initial research of the p5.js library, I developed a code plan and began sketching ideas.

First Code Plan: The game will be set up so that a person is walking back and forth at the bottom of the canvas. On the top portion of the canvas will be a cat that the user can move back and forth. This cat will be able to drop objects on the walking person when the user clicks on them. The user will gain points when the objects hit the person below.

Second Code Plan: The game allows to player to earn points by dropping objects on a pedestrian. The player can click anywhere on the canvas, which will trigger the cat to drop random office supplies. The player earns a point when the random office supplies collide with the pedestrian. On collision, the pedestrian walks faster which makes the game a little more difficult.
